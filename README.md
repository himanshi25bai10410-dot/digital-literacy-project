## Project Overview
This project demonstrates digital literacy skills through five tasks: creating an infographic, building a student digital portfolio, exploring coding and collaboration platforms, drafting professional emails and social media etiquette, and researching cybercrime awareness and prevention. Each task is documented with screenshots and reflections.

---

- **Tool used:** Canva  
- **Description:** Created a visually appealing Digital Literacy infographic covering its meaning, useful digital tools for students, safe internet practices, and maintaining a professional online presence. Learned to present information clearly and concisely for students.
